# Practice Site

A React-based practice website built with Vite and React Router.

## Features

- Multi-page navigation with React Router
- Custom cyberpunk-themed fonts
- Responsive design

## Pages

- Home
- About
- Contact
- Shop

## Tech Stack

- React 19.2.0
- React Router DOM 7.10.1
- Vite 7.2.4

## Getting Started

### Prerequisites

- Node.js installed on your machine

### Installation

1. Install dependencies:
```bash
npm install
```

### Development

Start the development server:
```bash
npm run dev
```

### Build

Create a production build:
```bash
npm run build
```

### Preview

Preview the production build locally:
```bash
npm run preview
```

## Project Structure

```
src/

├── fonts/          # Custom font files
```
